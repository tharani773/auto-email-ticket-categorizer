# Auto Email / Ticket Categorizer

## Description

This project classifies customer support tickets into Billing, Technical, HR, and General categories using Machine Learning.

## Technologies

- Python
- Pandas
- Scikit-learn
- TF-IDF
- Multinomial Naive Bayes

## Features

- Text preprocessing
- TF-IDF feature extraction
- Ticket classification
- Accuracy evaluation
- Classification report
- Confusion matrix
- Confidence score
- Human review detection
- Priority tagging
- Live CLI prediction
